# Dynatrace Configuration as Code Advanced samples


## dashboard-snippets

How to build dashboards with reusable parts (includes).

Deploys two dashboards:
- Dashboard Snippet 1
- Dashboard Snippet 2

Both use two snippets:
- JVM metrics for processes (by `processTag`)
- Service metrics for services (by `serviceTag`)


## dashboard-autolayout

How to automatically repeat same tile type with different parameters (tags) on a dashboard by a list. Includes both Classic and Gen3 Dashboard example.

Discussion on [Dynatrace Community](https://community.dynatrace.com/t5/Dynatrace-tips/PRO-TIP-Automatic-grid-layout-on-a-dashboard-with-monaco/m-p/284430)
